# File Organiser
A simple Python automation tool that organises files into folders based on file type.

## Features
- Automatically sorts files into folders
- Organises images, videos, audio, and documents
- Creates folders automatically
- Useful for project organisation and pipeline workflows

## How to use
Run the script:
```
python main.py
```

Enter the folder path, and the script will organise the files automatically.

## Example Structure
```
Folder/
    Images/
    Videos/
    Audio/
    Documents/
    Other/
```

## Why this tool
This tool was created as a simple workflow automation utility to help organise project files and maintain clean project directories.
